# librefranklin
Implementation of [7#551](https://github.com/unopengis/7/issues/551)

This is an experiment. You can use the Libre Franklin font as below: 

## glyphs
```json
"glyphs": "https://unopengis.github.io/librefranklin/{fontstack}/{range}.pbf"
```

## layout/text-font
```json
"layout": {
  "text-font": [
    "400"
  ]
}
```
